# NanoGraph_Renesas_RA8E1

Demonstration of NanoGraph on Cortex-M85 (RA8E1) : a prerecorded audio wave ([here](https://github.com/llefaucheur/NanoGraph_Computer?tab=readme-ov-file#nanograph-for-windows-computers)) is sent to a node (filter) and a detector triggers the LED of the board.
